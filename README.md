|Para este ejercicio se eligieron las columnas «país i», «país j», «col», «cor» y «lpn», donde «col» indica si el país_i y el país_j comparten al menos un idioma oficial, «cor» al menos un idioma oficial nacional estatutario o de facto, 1 si lo comparten y 0 si no lo comparten ambos, «lpn» es el índice de proximidad lingüística para diferentes idiomas nativos que se construye como $lpn_{ij} = \sum_k \sum_ {h\neq k}\frac{l_{ik}l_{jh}b_{kh}}{0.5(b_k+b_h)}$, donde $b_k$ es el número de ramas en el árbol genealógico lingüístico para la lengua k, b_{kh} el número de ramas que son comunes a las lenguas k y h, y l_{ki} denota la proporción de hablantes nativos de la lengua k en el país i. Para observar las relaciones menos obvias, se filtró el marco de datos estableciendo col = 0 y cor = 0, se imputó 0 a los valores faltantes y se filtró para lpn > 0. El marco de datos presenta algunas relaciones redundantes, como país_a - país_b con lpn = x, país_b - país_a lpn = x, por lo que se modificó el marco de datos para eliminarlas, lo mismo que con las relaciones país_i - país_i.  A continuación, se seleccionaron algunos países de Europa y la India, teniendo en cuenta que la mayoría de las lenguas europeas son indoeuropeas, para que fueran representativos y no repetitivos, por ejemplo, Austria, Alemania, Francia y Suiza. A continuación, se calculó la columna de distancia como 1, ya que el lpn se calcula como la suma y el producto de los hablantes compartidos por el lpn y las ramas comunes; un valor mayor indica una mayor proximidad lingüística entre las lenguas nativas y, por lo tanto, una menor proximidad cuando las distancias son mayores y viceversa. |For this exercise the columns ”country i”, ”country j”, ”col”, ”cor” and ”lpn” were chosen, where ”col” indicates if country_i and country_j share at least one official language, ”cor” at least one national statutory or de facto official language, 1 if they do and 0 if they don’t for both, ”lpn” is the Linguistic proximity index for different native languages which is built as $lpn_{ij} = \sum_k \sum_{h\neq k}\frac{l_{ik}l_{jh}b_{kh}}{0.5(b_k+b_h)}$, where $b_k$ is the number of branches in the linguistic family tree for language k, b_{kh} the number of branches that are common to languages k and h and l_{ki} denotes the share of native language speakers of language k in country i.
